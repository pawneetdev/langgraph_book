# LangGraph Book

A repository for learning and exploring LangGraph and LangChain concepts through practical examples.

## Notebooks

- [Hello World Agent](https://github.com/pawneetdev/langgraph_book/blob/master/notebooks/hello_world.ipynb) - LangGraph graph with a simple greeting node
- [Multiple Inputs Agent](https://github.com/pawneetdev/langgraph_book/blob/master/notebooks/multiple_inputs.ipynb) - LangGraph graph with multiple inputs
- [Sequential Agent](https://github.com/pawneetdev/langgraph_book/blob/master/notebooks/sequential_agent.ipynb) - LangGraph graph with sequential execution
- [Conditional Agent](https://github.com/pawneetdev/langgraph_book/blob/master/notebooks/conditional_agent.ipynb) - LangGraph graph with sequential execution
